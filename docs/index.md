## 什么是 ClawFlowGen？

ClawFlowGen 是一个基于"物理并行性"思想的处理器自动生成框架。它通过四阶段演化算法，从并行运算单元自动生长出完整的 CPU 或 NPU。

## 核心概念

1. **物理并行性**: 所有数字电路在物理上都是时刻并行的
2. **生长式生成**: 从算子孤岛到完整系统的四阶段演化
3. **元编程**: 使用 Python 动态生成硬件
